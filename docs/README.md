# Launcher Themes

You can either use the **builtin minimalist themes** (`Finns Theme` and `Dark Theme`) or create a **custom theme** from an image.

There are (almost ^^) endless possibilities for you to configure the theme, as it is **generated from your chosen image**.
Below you can also find some gorgeose [examples of custom themes](#custom).
